# Marvin Redpost 어휘·문법 (중2 대비)

동화 *Marvin Redpost* 를 읽기 전에 **핵심 단어와 문법**을 미리 익히는 중2 대비 학습 웹앱입니다. 단일 HTML로 동작하며(오프라인), 단어카드·말하기 시험·예문 시험·구문/문법 워크북·스마트 복습(SRS)·게임화 기능을 담았습니다.

## 사용법
`index.html` 을 브라우저에서 열면 됩니다. (별도 설치·서버 불필요)

## 구성
- **단어** — 책에 나오는 낱말 중 중2 수준을 선별, 세트별 단어카드(뜻·예문)
- **말하기 시험 / 예문 시험** — 음성·입력으로 채점, 관대한 한글 채점
- **구문·문법 워크북** — 어순 배열 / 어법 오류 수정 / 빈칸 어형 변화 / 조건 영작 (세트당 16문항)
- **스마트 복습(SRS)** · 데일리 챌린지 · 타임어택 · 뱃지 · 레벨/XP

## 저작권 안내
- 여기 실린 것은 **개별 낱말(어휘)** 목록이며, 원작의 본문(문장)·오디오는 포함하지 않습니다.
- **모든 예문·문법 문제 문장과 한글 뜻·해석은 학습용으로 새로 작성한 것**으로, 원작의 문장이 아닙니다.
- 개별 단어는 저작권 보호 대상이 아니며, 원작의 표현을 복제하지 않았습니다.
- 저작권은 원저작자에게 있으며, 본 자료는 어휘·문법 학습을 돕기 위한 비영리 교육용입니다.
