# Marvin Redpost 어휘 단어장

동화 *Marvin Redpost: Kidnapped at Birth* 에 나오는 **낱말(어휘)** 을 빈도순으로 추출해 만든 영어 단어 학습 웹앱입니다. 단일 HTML로 동작하며(오프라인), 단어카드·플래시카드·스마트 복습(SRS)·퀴즈·게임화 기능을 담았습니다.

## 사용법
`index.html` 을 브라우저에서 열면 됩니다. (별도 설치·서버 불필요)

## 구성
- **홈** — Day별 진행률, 연속학습·레벨·마스터리 대시보드
- **단어** — Day 1~8(단어 311개), 뜻·예문 카드, 외운 단어 체크
- **복습** — 망각곡선 기반 스마트 복습(Leitner 간격 반복)
- **퀴즈** — 뜻 고르기 / 뜻 입력(관대한 한글 채점) / 예문 빈칸
- **뱃지** — 학습 성취 뱃지 16종

## 저작권 안내
- 여기 실린 것은 **개별 낱말(어휘)** 목록이며, 원작의 본문(문장)·오디오는 포함하지 않습니다.
- **모든 예문과 한글 뜻·해석은 학습용으로 새로 작성한 것**으로, 원작의 문장이 아닙니다.
- 개별 단어는 저작권 보호 대상이 아니며, 원작의 표현을 복제하지 않았습니다.
- 저작권은 원저작자에게 있으며, 본 자료는 어휘 학습을 돕기 위한 비영리 교육용입니다.
